## Rule

If a decision changes the physical meaning, notation, normalization, data selection, code behavior, or manuscript content, it must be recorded here *before* changing the files.

## Decision types

- **Model choice**: which profile, which profile parameters, which approximation
- **Data choice**: which catalogue, which separation range, which quality cut
- **Notation change**: if a symbol is redefined or a convention is updated
- **Scope change**: if something previously in-scope is moved out of scope (or vice versa)
- **Error correction**: if a result is superseded by a corrected calculation
