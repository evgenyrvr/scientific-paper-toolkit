# Figures

Every figure file in this directory must have a corresponding provenance record.

## Provenance requirement

| Figure file | Source script | Data file | Run date | Parameters | Claims supported |
|-------------|--------------|-----------|----------|-----------|-----------------|
| [figure.pdf] | [code/script.py] | [data/output.dat] | YYYY-MM-DD | [key params] | CL-NNN |

## Rules

1. No figure may be included in the manuscript without a row in the table above.
2. No figure parameter may be changed without updating the row and recording the reason.
3. Run logs must be stored in the `code/` or `iterations/` directory.
