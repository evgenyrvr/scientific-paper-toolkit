# Research Question Scope

## Purpose

Turn an idea into a precise, falsifiable, publishable research question.

## Required files

- `project/01_research_question.md`
- `project/02_assumptions_and_conventions.md`
- `project/03_claims_register.md`
- `project/04_risk_register.md`

## Procedure

1. Extract the central physical system, quantity, phenomenon, or equation.
2. Write a one-sentence research question.
3. Write a scoped version with assumptions and exclusions.
4. Identify candidate novelty claims.
5. Separate:
   - what is known;
   - what is hypothesized;
   - what is derived;
   - what is numerically tested;
   - what is speculation.
6. Add each candidate claim to `project/03_claims_register.md` with status `draft`.
7. Add ambiguous or risky points to `project/04_risk_register.md`.
8. Create an iteration report.

## Required output sections

- Problem statement.
- Physical context.
- Target audience.
- Candidate contribution.
- Assumptions.
- Out-of-scope list.
- Risks.
- Next literature tasks.

## Red flags

Mark `BLOCKED` if:

- the main quantity is not defined;
- the result cannot be falsified or checked;
- the novelty is only rhetorical;
- required prior literature is unknown.
